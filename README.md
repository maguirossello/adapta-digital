# Adapta Digital

Sitio institucional del proyecto **Adapta Digital**, consultora de transformación digital para PyMEs no tecnológicas basada en Mar del Plata.

Trabajo Práctico Integrador · Gestión de Empresas de TI · Universidad FASTA · 2026
Rossello, María Magdalena

## Contenido

- **`index.html`** — Sitio institucional completo (versión para lectura)
- **`presentacion.html`** — Versión reducida para presentación oral en vivo
- **`img/`** — Imágenes, logos y dashboards del sitio

## Uso

Abrí `index.html` en cualquier navegador para explorar el sitio institucional.
Abrí `presentacion.html` en pantalla completa (F11) para la presentación oral — navegación con flechas del teclado (← →) o click en el menú.
